Tech Sharing - JS Sort Visualization

  In the previous article I talked about a story about why do I start to do the sort visualization thing. In this article, I will talk about some tech thing about sorting algorithms.

  So for the first thing I would like to talk about is sorting. In the demo sort visualization, I showed four sorting algorithms. Bubble sort, Selection sort, Exchange sort and Insertion sort. These sorting algorithms will be taught when students are freshman at NDHU. To be honest, these sorts aren't hard at all. However, sorting is too difficult for those who are exposed to programming languages for the first time, so, I wrote this to let them know more by what is sorting.

  Let's have a brief look at the four sorting algorithms. Starting with bubble sort.

  Bubble sort is a simple sorting algorithm that repeatedly steps through the list, compares adjacent elements and swaps them if they are in the wrong order. The pass through the list is repeated until the list is sorted.

  So the code of bubble sort will be: 

                    
//C++ bubble sort
for(int i = n - 1; i > 0; i--){
    for(int j = 0; j < i; j++){
        if(arr[i] < arr[j]){
            swap(arr[i], arr[j]);
        }
    }
}

  That's not to hard to understand, right? Now, it's time for the other three sorting algorithms.

  For selection sort, we have a target that goes through the whole array, choosing the element that is the smallest after the target. After the go through the array, swap the two elements. Let's take [5, 2, 3, 1, 9] for example.

  For the array [5, 2, 3, 1, 9]. Our target is 5, the compare the smallest element after 5, it's 1. So swap 1 and 5 makes the array [1, 2, 3, 5, 9]. I will leave the last compare loop for you.

  On top of that, we can figure the code of selection sort will be: 

                    
//C++ selection sort
for(int i = 0; i < n; i++){
    int temp_index = i;
    for(int j = i + 1; j < n; j++){
        if(arr[j] < arr[temp_index]){
            temp_index = j;
        }
    }
    if(i != temp_index){
        swap(arr[i], arr[temp_index]);
    }
}

  For exchange sort, for me, it's more easy to understand. Starting from the beginning, the target compares all the elements, swap them if the element is smaller than the target. (If you want array is sorted in ascending order) The code is really simple as well: 

                    
//C++ exchange sort
for(int i = 0; i < array.length(); i++){
    for(int j = i + 1; j < array.length(); j++){
        if(array[i] > array[j]){
            swap(array[i], array[j]);
        }
    }
}

  For insertion sort, you can think that insertion sort is simillar to ordering a poker card. Choose a card from the first element of the array and insert the card to somewhere until all the cards are sorted. The code of insertion sort are as follows: 

                    
//C++ insertion sort
for(int i = 1; i < n; i++){
    int key = arr[i];
    int j = i - 1;
    while(key < arr[j] && j >= 0){
        arr[j + 1] = arr[j];
        j--;
    }
    arr[j + 1] = key;
}
